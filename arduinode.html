Arduinode

Aplicación web para el control de distintos dispositivos de un hogar (Luces, persianas, sistemas de riego). Hecho en React, NodeJS y Socket.IO // Arduinode is a home automation system. It lets you switch the lights of the house, control the windows and the irrigation system. Also, it can schedule operations to automate some tasks. Built on top of Node.JS, Socket.IO, React.JS and Arduino

Detalles // Details

  • Control de luces, persianas, sistemas de riego. // Lights, windows and irrigation system control.
  • Tareas programadas // Scheduled tasks
  • Temporización de dispositivos // Timer actions
  • Control por comandos de voz // Control over voice commands
